# Hera BMAD Bridge

This repository is intentionally BMAD-native. The project process is:

1. Install BMAD.
2. Ask `bmad-help` for the next workflow.
3. Run the recommended BMAD workflow through a coding tool adapter.
4. Let BMAD artifacts/code update.
5. Hera verifies diff, tests, safety boundaries.
6. Ask `bmad-help` again.

Hera does not replace BMAD. Hera operates BMAD.
